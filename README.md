## Current focus

- **Super Agent infrastructure** — Hermes, Agent Zero, Codex, OpenRouter, Telegram, Slack, Obsidian, and repo-backed operating memory.
- **AI Integraterz** — productized AI implementation offers for staffing, recruiting, home services, experts, and local businesses.
- **SpeakerAgent.ai** — ops agents, GTM systems, LinkedIn authority, and execution dashboards for speaker-led businesses.
- **Agent companies** — multi-agent teams that behave like a small department instead of a chatbot.
- **Content and authority engines** — turning one expert input into campaigns, posts, scripts, emails, and offers.
